# Roadmap

## Sprint 1 (2 weeks)
- Auth service (passkeys, rotating refresh tokens, session reuse detection)
- Key service (bundle upload/fetch/claim one-time prekeys)
- Messaging service (ciphertext envelope persistence)
- Gateway (AUTH, SUBSCRIBE, MESSAGE_CREATED)
- Web shell UI + design tokens

## Sprint 2
- Server/channel CRUD + permissions engine
- Encrypted attachments
- Presence + typing + receipts
- Audit/security event pipeline
- AI v1: summarize, catch-up, draft assist (strict ACL scoped)

## Sprint 3
- Hardening and scale
- Threat-model validation and red-team fixes
- Admin/security console
- Beta readiness checklist
